EcoBin - Smart Waste Management

EcoBin is a smart waste management system that uses an ESP32 microcontroller with ultrasonic sensors to monitor the trash level and control the trash can lid via a servo motor. It connects to WiFi and integrates with the Blynk app for real-time monitoring of trash levels and remote notifications when the bin is full. An OLED display shows the trash level and WiFi status. The system automatically opens or closes the lid based on object detection and sends alerts if the bin is near full capacity.

Features

Trash Level Monitoring

An ultrasonic sensor measures the distance to the trash, calculating the trash level. This data is displayed on an OLED screen and sent to the Blynk app for remote monitoring.

Lid Control

A second ultrasonic sensor detects nearby objects, automatically opening the trash can lid when within 30 cm and closing it when no one is near, promoting hands-free operation.

Wi-Fi Connectivity

The ESP32 connects to Wi-Fi, enabling the trash level data to be sent to the Blynk app. Notifications are triggered when the trash level reaches 90 to 100%, helping users stay informed.

Blynk Integration

In the Blynk app will monitor the trash level and the lid with notifications to the user when the trash can is full.

Materials

ESP32 Microcontroller

The heart of EcoBin, the ESP32 handles all the processing tasks, including sensor reading and Wi-Fi communication.

Ultrasonic Sensors (HC-SR04)

Used for measuring trash level and detecting proximity for automatic trash can lid operation.

Servo Motor (SG90)

The servo motor controls the opening and closing of the trash can lid, triggered by proximity sensors.

OLED Display (SSD1306)

Displays system status, including Wi-Fi connectivity and trash level for the EcoBin system.

Wi-Fi Router

Provides the internet access needed for the ESP32 to send real-time updates and communicate with the Blynk app.

Blynk App

The Blynk app allows users to monitor and control the EcoBin remotely, providing real-time updates and alerts.

Jumper Wires

Essential for connecting the ESP32 with sensors, motors, and other components in the EcoBin system.

Plywood

Used to construct the housing and body of the EcoBin, providing durability and a lightweight structure.

How It Works

EcoBin uses IoT technology to monitor trash levels through sensors, sending data to the Blynk Cloud via Wi-Fi. Users can view bin status and receive alerts when the bin is full. This system ensures efficient waste management by providing real-time updates for timely maintenance and optimized usage.

Applications

Home

The system can be used in households to automatically open the trash can lid when a person is near and monitor the trash level. Notifications are sent when the trash is full, making trash management more convenient and reducing manual effort.

Office

In offices, the system helps maintain cleanliness by notifying staff when trash cans are full and automates the lid opening to avoid physical contact with the bin, promoting hygiene and reducing the need for constant monitoring.

School

In schools, this system helps track trash levels in classrooms or cafeterias. It automatically opens the trash can lid for students and sends alerts when bins are full, reducing the need for frequent manual checks by staff.
